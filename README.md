Aplicação web desenvolvida em **React + Vite** que consome a API oficial de **Dungeons & Dragons (5e)** para listar, organizar e traduzir conteúdos do jogo.

> 🧙‍♂️ Um companion digital para facilitar a vida de jogadores e desenvolvedores.

## ✨ Funcionalidades

- 📡 Integração com API oficial de D&D 5e
- 📚 Listagem de conteúdos:
  - Classes
  - Subclasses
  - Raças e Sub-raças
  - Magias
  - Equipamentos
- 🌐 Tradução (PT-BR e ES)
- 🧩 Organização e padronização dos dados
- 🎨 Interface moderna com ícones (Lucide)
- ⚡ Alta performance com Vite

---

## 🛠️ Tecnologias

- ⚛️ React
- ⚡ Vite
- 🟦 TypeScript
- 🎨 Lucide Icons

## 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/dungeon-companion.git

# Acesse a pasta
cd dungeon-companion

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```
